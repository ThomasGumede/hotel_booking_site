# Next.js + Tailwind CSS Hotel Booking Website

Landing page for hotel booking website using Nextjs & JavaScript

It uses the new [`Just-in-Time Mode`](https://tailwindcss.com/docs/just-in-time-mode) for Tailwind CSS.
